# apply-page-numbers
Apply page numbers to all the page of the PDF!

## **Usage**
```shell
python3 apply-page-number.py <path of  original file> <path of the new file>
```
To see more options, run the script
```shell
python3 apply-page-number.py
```

## **Dependencies**
Self Installed
